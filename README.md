🎮 Rock Paper Scissors Game using Python

📖 Overview

This project is a simple command-line based Rock Paper Scissors game developed using Python.
The user plays against the computer, where the computer randomly selects one option from Rock, Paper, or Scissors.
The program then compares both choices and declares the winner according to the game rules.

---

✨ Features

- Interactive user input
- Random computer-generated moves
- Winner detection logic
- Tie condition handling
- Invalid input handling
- Clean and beginner-friendly code structure

- Random Module

🎯 Game Rules

Rock| Scissors| Rock Wins
Rock| Paper| Paper Wins
Paper| Rock| Paper Wins
Paper| Scissors| Scissors Wins
Scissors| Paper| Scissors Wins
Scissors| Rock| Rock Wins
